# MCBE ProxVC

[![Wiki](https://img.shields.io/badge/Wiki-Documentation-blue?style=for-the-badge&logo=read-the-docs)](https://mcproxvc.ovh/en/wiki/)
[![Download](https://img.shields.io/badge/Download-Latest_Release-green?style=for-the-badge&logo=minecraft)](https://github.com/kkrkun/MCBE-ProxVC/releases/latest)
[![Discord](https://img.shields.io/badge/Discord-link-blue?style=for-the-badge&logo=discord&logoColor=white)](https://discord.com/invite/bQcFBKnhSQ)

An addon that adds **Proximity Voice Chat** and **Proximity Text Chat** features to Minecraft Bedrock Edition.
It creates an immersive conversation experience based on distance, similar to mods for the Java Edition.
